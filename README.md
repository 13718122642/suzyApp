# suzy.live App

> 使用react-native + redux开发的App

##功能
下拉刷新
上拉加载更多
闪屏放大效果
文章列表
添加文章
添加评论
点赞

## 技术栈
react-native
redux

##第三方组件
react-native-image-picker  拍照功能
react-native-loading-spinner-overlay 加载中遮罩
react-native-vector-icons 应用各种icon

## 安装步骤

``` bash
# 安装依赖
npm install

# 运行应用
react-native run-ios
```
